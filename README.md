# Pomodoro Planner

A simple Pomodoro timer app with planning features.

## Features
- Start and stop Pomodoro sessions
- Track planned tasks
- Save your progress locally

## Run
Run the app with:

```bash
python main.py
```

## Notes
This project is a lightweight personal planner for managing focus sessions.
