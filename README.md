# Pomodoro Planner

A lightweight Pomodoro timer and task planner built with Python and CustomTkinter.

## Features

- Pomodoro Timer
- Planner
- JSON-Based Local Data Saving

## Data Saving

Planner and Pomodoro data are stored locally in AppData as JSON files.
